# Privacy Policy for DNR Domain Redirector

Last updated: 2025-02-08

## Data Collection

DNR Domain Redirector does NOT collect, store, or transmit any personal data.

## Local Storage

The extension stores your redirect rules locally in your browser using the browser.storage.local API. This data:
- Never leaves your device
- Is not transmitted to any servers
- Is not shared with the developer or any third parties
- Is only used to apply your configured redirect rules

## Permissions Explanation

- **declarativeNetRequest**: Required to redirect HTTP/HTTPS requests based on your rules
- **declarativeNetRequestWithHostAccess**: Required to access domain information for matching redirect rules
- **storage**: Required to save your redirect rules locally in your browser
- **host_permissions (<all_urls>)**: Required to match and redirect any domain you configure. The extension only acts on domains you explicitly add to your redirect rules.

## No External Communication

This extension:
- Does not make any network requests
- Does not load any remote code
- Does not use any analytics or tracking
- Does not contain any advertisements

## User Control

You have complete control over your data:
- All redirect rules are managed through the extension's options page
- You can delete any or all rules at any time
- Uninstalling the extension removes all stored data

## Changes to This Policy

Any changes to this privacy policy will be posted in the extension's documentation.
